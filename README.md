# Project-D-Portfolio

**장르:** 연구소 배경의 바이오 호러 컨셉 변신 액션 RPG  
**플랫폼:** PC  
**개발 툴:** Unity 6  
**개발 기간:** 2025년 7월 1일 ~ 1월 10일 (약 27주)

---

##  프로젝트 개요

Project:D는 플레이어가 '포식자'로서 몬스터의 능력을 흡수하고 성장하는 로그라이크 RPG입니다. 실험실에서 탈출한 주인공이 순환하는 던전 구조 속에서 각성하며, 점점 강해지는 전투와 스토리를 경험하게 됩니다.

---

##  개발 목표

- **데모 구현 범위:**  
  - 던전 2층까지 플레이 가능  
  - 포식 시스템 체험 가능

---

##  핵심 시스템 및 기능

### 1. 포식 시스템
- 몬스터 처치 시 **포식 가능**
- 포식 시 해당 몬스터의 **스탯 또는 스킬** 흡수
- 일정 조건 충족 시 **계열 각성**
  - 각성 시 전투 모션/스킬 변화
  - 계열별 전투 스타일 분기 가능

### 2. 던전 구조 및 보스 설계
- 층별로 **보스 및 패턴 구성**
- 던전은 반복적으로 도전 가능한 **순환형 구조**
- 보스를 처치하면 다음 층으로 진행 가능

### 3. 각성 UI
- 체력바 옆에 현재 **각성 계열** 시각화
- 전투 중 실시간으로 각성 상태 확인 가능

### 4. 스토리 흐름
- **시작:** 연구소/실험실 탈출  
- **중반:** 포식과 각성을 통해 자아 확립  
- **후반:** 실험의 루프 구조를 자각하며 탈출 방법 탐색

---

##  기술 스택

- Unity 6 (2025 기준 최신 LTS)
- C# (Gameplay, 시스템 로직)
- DOTween Pro (UI/애니메이션 처리 예정)
- GitHub + Git LFS (버전 관리 및 협업)

---

##  팀 목표

- **완성도 높은 1~2층 던전과 전투 구현**
- **포식 → 각성 → 성장 흐름의 재미 전달**
- **기본 UI와 애니메이션 정제**
